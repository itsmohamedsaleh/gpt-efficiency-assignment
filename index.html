Introduction

This project, ChatGPT Validator, aims to evaluate how well ChatGPT performs in different areas. We built a website that connects to a database and uses ChatGPT to answer questions. By comparing ChatGPT's answers to the correct ones, we can see how accurate and efficient the model is.

The workflow includes several key steps:


  1. Database Setup: Using the terminal and Visual Studio Code (VSCode), we created a MongoDB database to store our data.
  2. Backend Development: We used Node.js and Express to connect to the MongoDB database, allowing us to manage data effectively.
  3. API Integration: We integrated the ChatGPT API to generate responses based on user questions.
  4. Data Upload: Three CSV files with questions and answers were uploaded to populate the database.
  5. Response Collection: ChatGPT's answers to the questions were stored in the database for comparison.
  6. Performance Analysis: We compared ChatGPT's answers to the correct ones to measure accuracy and response time.
  7. Results Display: The results were shown using charts and statistics to highlight ChatGPT's performance.

This project not only shows how ChatGPT works but also provides a way to measure its effectiveness in different subjects.

Research Approach

The research for the ChatGPT Validator project followed a clear and organized method to ensure accurate results. Here are the main steps we took:


  • Setting Up the Environment: We used Visual Studio Code (VSCode) to start the project and set up all necessary tools.
  • Creating the Database: Through the terminal, we set up a MongoDB database to store questions, answers, and ChatGPT responses. We defined the database structure to keep the data organized.
  • Developing the Backend: Using Node.js and Express, we built the server-side application. This handled API requests, managed the database, and connected the frontend with the backend.
  • Integrating the ChatGPT API: We connected to the ChatGPT API using the `chat.completion` endpoint. This allowed our application to send questions to ChatGPT and receive answers automatically.
  • Uploading Data: We uploaded three CSV files containing a variety of questions and their correct answers. These were imported into the MongoDB database.
  • Collecting ChatGPT Responses: For each question in the database, we sent a request to ChatGPT and saved its response in the database.
  • Comparing Answers: We compared ChatGPT's answers with the correct ones to measure accuracy and response time.
  • Visualizing Results: The comparison results were displayed using charts and statistics to make the data easy to understand.

This structured approach helped us thoroughly evaluate ChatGPT's performance and provided clear insights into its strengths and areas for improvement.
