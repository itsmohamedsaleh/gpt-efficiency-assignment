Introduction

The aim of this project is to develop an interactive platform that would evaluate and present ChatGPT's performance in different contexts.

The site has several objectives: to showcase the web development aspects of a data-oriented and customer-friendly platform and to highlight the capabilities of ChatGPT, in a conversational setting. With the help of data from various disciplines such as history, social science, and Computer security, this site enables users to assess the effectiveness and the weaknesses of ChatGPT when used in various domains.

The project utilizes functions of both frontend web application development and backend data processing. Technologies applied include but are not limited to MongoDB, Node.js, and the ChatGPT API to present the outcomes in a way that is broad appeal, interesting, and useful.

Research Approach

We had to test the performance of Chatgpt, so we developed an interactive website based on front-end and back-end technologies. The primary database for storing data related to the evaluations including the questions, the answers, and the scoring metrics is MongoDB. The backend is built upon Node.js, allowing everything from data management to embedding the ChatGPT API. It is this API that provides the questions and generates answers that, in turn, are used to evaluate the accuracy, relevance, and consistency of ChatGPT. 'Pods' can be used to present the interface of this website to the visitors and in turn, it allows its visitors to test the outcomes and observe the site's ability, performance, and weaknesses of Chatgpt.

Dataset Overview

The dataset consists of a tailored set of questions covering three areas, in particular: History, Social Science, and Computer Security. This variety guarantees that the overall assessment of ChatGPT is exposed to a versatile outlook regarding topics and knowledge domains. Each of the questions was formulated to assess the different elements of the model’s capabilities such as functional correctness, comprehension, and even level of detail. This examination aims to indicate the scope of applicability of the ChatGPT system in different disciplines showcasing its strengths and weaknesses.
